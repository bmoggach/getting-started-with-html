The Basics of the Web

The web is collection of 30 billion pages. These pages are HTML documents that can contain text, music, images, videos, virtually anything!

Some basic elemets of the web include:
  1. hypertext markup language - the basis for a webpage
  2. URLs - links that make the web… weblike.
  3. HTTP - the protocol the browser uses to communicate to servers

Major pieces of the web
  1. You, your computer, and the browser your computer is using
  2. The Internet
  3. Servers - optimized file hosting computers.
This all works in the following way: you make requests to the internet via servers.. server responds with files that the browser displays HTML markup is made up of tags, attributes, and elements. All tags require a closing or end tag.. EXCEPT for a void tag. It does not require a closing tag.

Components of HTML

  • Attributes always equal a value. Example of a attribute is an anchor tag… <.a.>... used to create a link. <.a href=””.> <./a.>
  • Element is everything within a set of tags.
  • For an image, use <.img src=”” alt=”text”.> … alternate displays text on broken requests, no end tag.. example of a void tag.
  • <.b.> bold
  • <.em.> italicized
  • <.p.> paragraph

    • Inline vs. Block Elements

    Block elements create some white space around a text.. for example whereas inline elements do not

    <.span.> is an inline element and <.div.> is a block element
